# NomenGlobe : Nom & Prénom explorateur

Ce projet est une application d'analyse de données qui permet d'obtenir des informations détaillées sur n'importe quel nom ou prénom.

## 🚀 Fonctionnalités
- **Origine géographique :** Identifie la provenance historique du nom.
- **Rang mondial :** Détermine la position du nom dans le classement global.
- **Statistiques de porteurs :** Affiche le nombre total de personnes portant ce nom.
- **Fréquence :** Analyse la rareté ou la popularité du nom.

## 🛠️ Stack Technique & Bibliothèques
L'application est construite avec **Python 3.12** et s'appuie sur les écosystèmes suivants :

- **Interface & Visualisation** : Streamlit, Plotly Express (pour les graphiques dynamiques).

- **Backend & API** : FastAPI, Uvicorn, Requests.

- **Data Processing** : Pandas, NumPy.

- **Web Scraping & Automation** : Playwright (Async/Sync), BeautifulSoup4.

- **Data Matching & Qualité** : RapidFuzz (levenshtein/fuzzy matching), Scikit-learn (métriques de performance).

- **Utilitaires** : TQDM (barres de progression), Unicodedata, Re (Regex).
